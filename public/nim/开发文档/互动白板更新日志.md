# Web 端更新日志

## [5.0.0] - 2018-03-29

### 新增

- 绘图新增接口 setContainer 支持动态设置canvas容器，适用于单页应用
- 绘图新增接口 enableDraw 支持动态设置是否允许绘图动作，默认允许
- 绘图新增接口 image 支持动态绘制背景图片
- 绘图新增接口 clearImage 清空绘制的背景图片
